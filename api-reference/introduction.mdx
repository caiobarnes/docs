## 📝 Detalhamento das Regras

<Accordion title="🚫 Divulgar Servidor">
  ### O que é considerado divulgação?

  Compartilhar IP, links ou fazer propaganda de outros servidores. Seja diretamente enviando o endereço ou indiretamente sugerindo que os players joguem em outro lugar.

  #### Exemplos:

  * Enviar links de outros servidores

  * Compartilhar IP de outros servidores

  * Fazer propaganda indireta

  * Tentar recrutar jogadores
</Accordion>

<Accordion title="💬 Mencionar outros servidores">
  ### O que não é permitido?

  Falar sobre outros servidores, mesmo sem divulgar IP. Como comparar servidores, falar de eventos de outros lugares ou tentar levar players para outra comunidade.

  #### Evite:

  * Comparações entre servidores

  * Mencionar eventos externos

  * Discussões sobre outros servidores

  * Sugerir outros lugares para jogar
</Accordion>

<Accordion title="⌨️ Spam Comando">
  ### O que é spam de comando?

  Usar comandos repetidamente para floodar o chat ou causar incômodo. Seja spam de /tpa, /anunciar ou qualquer outro comando que polua o chat.

  #### Comandos mais abusados:

  * `/tpa` repetidamente

  * `/anunciar` sem necessidade

  * `/tell` em excesso

  * Qualquer comando usado para perturbar
</Accordion>

## 💡 Por que essas regras são importantes?

* 🎯 **Foco na Comunidade**: Mantemos nossa comunidade unida

* 🛡️ **Proteção**: Evitamos a dispersão de jogadores

* 📝 **Organização**: Prevenimos poluição no chat

* 🤝 **Respeito**: Promovemos respeito entre todos
